# dotfiles-win
My Bash on Windows configuration files.

### usage
Just run `install.sh` for pull last git version and symlink all dotfiles.

```
$ git clone https://github.com/jieverson/dotfiles-win.git ~/.dotfiles
$ ~/.dotfiles/install.sh
```
